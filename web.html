POKEDER

Pokeder est un tinder like qui permet de trouver des personnes qui aiment le même type de Pokemon que vous. Une fois l'inscription faite, une liste de personnes est suggérés. L'utilisateur peut choisir de liker ou non l'utilisateur proposé.

Création d'une API pour le like et l'archivage. Utilisation de l'API PokéAPI et de l'API Giphy

Outils : HTML, CSS, JavaScript, AJAX, PHP, SQL

Effectué avec Lucie Lesbats

SPOTTIMAC

Spottimac est un site web lié à l’IMAC qui permet de poster des textes, photos ou vidéos sous forme de publication grâce à un compte que l’utilisateur se créer.

Chaque publication est liée à une catégorie avec des thèmes et des sujets appropriés. Les utilisateurs peuvent liker et/ou commenter chaque publication dès qu’ils sont connectés.

Outils : HTML, CSS, JavaScript, AJAX, PHP, SQL
